# CommentTree

Сервис для работы с древовидными комментариями с неограниченной вложенностью, полнотекстовым поиском и веб-интерфейсом.

## Возможности

- Создание комментариев с указанием родительского
- Получение дерева комментариев с постраничной навигацией
- Удаление комментария вместе со всеми вложенными ответами
- Полнотекстовый поиск по содержимому комментариев
- Ограничение максимальной глубины вложенности
- Веб-интерфейс с визуальной вложенностью

## Архитектура
Дерево комментариев хранится с помощью PostgreSQL-расширения **ltree**. Каждый комментарий имеет поле `path` типа `ltree`, которое содержит путь от корня до текущего узла (например, `1.4.6.7`). Это позволяет:

- Получить всё поддерево одним запросом: `WHERE path <@ '1'::ltree`
- Удалить поддерево атомарно
- Сортировать в DFS-порядке: `ORDER BY path`

## Запуск

### Docker Compose

```bash
git clone https://github.com/stpnv0/Comment-Tree.git
cd CommentTree

docker compose up -d --build
```
Приложение будет доступно по адресу: http://localhost:8080
